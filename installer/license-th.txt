ขอบคุณที่เลือกใช้ HAPPY ❤️ โปรดอ่านข้อตกลงสั้นๆ ก่อนติดตั้ง

1. สิทธิ์การใช้งาน
   - HAPPY แจกฟรีสำหรับการใช้งานส่วนตัวและการเรียนรู้
   - ไม่อนุญาตให้นำไปขายต่อหรือดัดแปลงเพื่อการค้า
     โดยไม่ได้รับอนุญาตจากผู้พัฒนา (Nick SuksanTr)

2. ความรับผิดชอบของผู้ใช้
   - ต้องมี Gemini API key ของตัวเอง (ฟรีหรือเสียเงิน)
   - ค่าใช้จ่ายของ AI API เป็นความรับผิดชอบของผู้ใช้
   - HAPPY ไม่เก็บข้อมูล / ไม่ส่งข้อมูลของคุณออกไปนอกเครื่อง
     (ยกเว้นที่ต้องส่งไปยัง Gemini API เพื่อทำงาน)

3. การรับประกัน
   - ซอฟต์แวร์นี้แจก "ตามที่เป็นอยู่" (as-is)
   - ผู้พัฒนาไม่รับผิดชอบความเสียหายจากการใช้งาน
   - โปรดทดสอบในสภาพแวดล้อมที่ไม่ critical ก่อน

4. การอัปเดต
   - HAPPY ไม่มี auto-update — ดาวน์โหลดเวอร์ชันใหม่จาก
     https://github.com/nicksuksantr/happy-ai-agent
   - การอัปเดตเป็นทางเลือก ไม่บังคับ

5. การยกเลิก
   - ถอนการติดตั้งได้ทุกเมื่อจาก Settings > Apps
   - API key ของคุณจะยังอยู่ใน ~/.happy/auth.json
     สามารถลบเองภายหลังถ้าต้องการ

# user/ — ไฟล์ส่งออกให้ผู้ใช้

- ภาพหน้าตาแอป (screenshots)
- ตัวอย่าง output / mockup
- ไฟล์ส่งมอบอื่น ๆ

(ตาม Note Master + command_pattern ข้อ 6 — อย่าวางไฟล์พวกนี้กระจัดกระจายใน root)
